# UI Plan

## Visual direction
- Dark theme with bright accent colors.
- Gaming look without making it feel too busy.
- Main focus should stay on products, prices, and store actions.

## Main pages

### Home page
- Short intro to the site.
- Quick stats.
- Featured products.
- Platform section with PS5, Xbox, Switch, and PC.

### Auth pages
- Sign up
- Login
- Password reset flow

### Vendor pages
- Dashboard
- Create store page
- Edit store page
- Create product page
- Edit product page

### Buyer pages
- Product list
- Product detail
- Cart
- Checkout
- Order detail

## Layout choices
- Use a top navigation bar so users can move around fast.
- Keep store and product actions close to the main content area.
- Show review status clearly so verified reviews stand out.
- Keep forms simple and not too long.

## Mobile thoughts
- Buttons need to stay easy to tap.
- Product cards should stack instead of squeezing into tiny columns.
- The cart and checkout pages should keep totals easy to read on small screens.
